# PM_Tracker

Preventive maintenance tracker for hospital biomedical teams with reminder automation, communication history, and contract visibility.

## Current capabilities

- PM equipment tracking by hospital/department/engineer.
- CSV/Excel import and CSV/JSON export.
- Hospital status view with ready-to-send email draft.
- COM history per hospital equipment (comments + email history + PM history).
- Contract reminder view.

## Scripts

- `npm run dev` - run web app locally.
- `npm run build` - build production web app.
- `npm run test` - run node tests.
- `npm run desktop:dev` - run Tauri desktop shell.
- `npm run desktop:build` - build Tauri desktop package.
